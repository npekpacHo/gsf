Инструмент создания локальных фавиконок для GSF (ИПИГ)

Файл скприпта gsf.user.js должен лежать в папке.
После запуска RUN.bat (от имени Администратора) скрипт будет распарсен и для прописанных доменов в папке images будут созданы изображения.
*images отсутствует? - будет создана
*сущетсвующие изображения не будут замененены

После завершения работы изображения из images следует загрузить в одноименную папку на Гитхаб.
